\# Программа управления меню 🍽️



Лабораторная работа по дисциплине "Технология и методы программирования"

Вариант 4: Меню (название, цена, время приготовления, цвет)



\## Описание программы



Программа позволяет управлять меню ресторана:

\- Добавлять блюда с указанием названия, цены, времени приготовления и цвета

\- Просматривать список всех добавленных блюд

\- Цвет блюда является опциональным параметром



\## Формат ввода данных

"Название блюда" цена время(чч:мм) \[цвет]



Примеры:

\- `"Борщ" 5.75 01:30` - без указания цвета

\- `"Суп" 6.21 02:12 красный` - с указанием цвета



\## Функционал



\- \*\*1\*\* - Добавить новое блюдо

\- \*\*2\*\* - Показать всё меню

\- \*\*stop\*\* - Завершить работу программы
